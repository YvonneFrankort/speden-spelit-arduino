# Speden Spelit – Arduino Game (Embedded Systems Project)

A team-based embedded systems project developed as part of the **Application Project in Information Technology** course.

This project combines hardware and software to implement an interactive memory-style game using an Arduino microcontroller, LEDs, buttons, and 7-segment displays.

---

## Overview

The goal of the project was to design and implement a modular Arduino application that integrates:

- Physical button inputs
- LED visual feedback
- Dual 7-segment score display
- Timer-based game logic and random sequence generation

The system was built using C++ for Arduino with a clear modular structure separating hardware control and game logic.

---

## Key Features

- Modular architecture (Buttons, LEDs, Display, Game Logic)
- Timer-based random number generation
- Interrupt-driven input handling
- Real-time score display on 7-segment displays
- Hardware + software integration

## How to Run

1. Open SpedenSpelit.ino in the Arduino IDE.
2. Connect the Arduino to the circuit according to the project documentation.
3. Upload the sketch and start the game.
